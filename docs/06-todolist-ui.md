## 里程碑目标

浏览器体验接近 VIGI Web 的信息架构与交互密度；功能覆盖本板已支持或短期可落地的能力；UI 统一使用 **Vue3 + Element Plus（按需）**。

验收总标准：

1. 登录后顶栏：预览 / 信息 / 设置；设置内左侧树可到达全部已实现页
2. 移动侦测：预览上画区域 + 布防时间表 + 联动录像/抓图，走动可出事件与抓图
3. 存储：可配事件/定时计划，可回放，可看容量与软清空
4. `www/` 构建通过；板端硬刷新可用；包体未压缩 &lt; 1.5MB（尽量）

## 明确不在本 Todolist

| 项 | 原因 |
|---|---|
| PTZ / 预置位 / 巡航 / 光学变焦 | 无硬件；用户排除 |
| 音频 / 对讲 / 声光报警 | 无外设 |
| 越线/入侵/LPR/人形车辆（无模型） | 依赖 NPU/算法，见 T3.4 / 二期 |
| ONVIF/邮件/FTP/DDNS/SNMP/P2P/Wi‑Fi | 非本板产品范围 |
| 真卡格式化 mkfs、在线升级 | 风险高，另开里程碑 |
