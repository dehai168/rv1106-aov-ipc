# T1.3 + M2 设计要点

## T1.3
- `media_stream_start/apply/stop`：常驻双码流；apply 时按需重建 VI/VENC（分辨率变则重建 VI+VENC，仅码率/GOP/帧率则重建 VENC）
- CLI：`ipc_app --reconfig` 先录高码率 5s，再 apply 低码率录 5s，对比文件大小

## M2
- TF 已是 exFAT 挂载 `/mnt/sdcard`（板上确认）
- 录像用板上 `librkmuxer`（支持 H.265→MP4），不用 minimp4
- 目录：`/mnt/sdcard/records/YYYYMMDD/YYYYMMDD_HHMMSS.mp4`
- `ipc_app --record [seconds]`：分段录制 + 循环覆盖 + 索引查询自检
